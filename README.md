# HumanX Lab Website

Website for the HumanX Lab at Dartmouth College.

## Structure

```
humanx-website/
├── index.html
├── people.html
├── projects.html
├── contact.html
├── recruitment.html
├── assets/
│   ├── images/
│   │   ├── logo.png
│   │   └── dartmouth-logo.png
│   └── css/
│       └── style.css
├── scripts/
│   └── main.js
└── README.md
```

## Setup

1. Add your logo image to `assets/images/logo.png`
2. Add the Dartmouth logo to `assets/images/dartmouth-logo.png`
3. Customize the HTML files with your content
4. Update styles in `assets/css/style.css` as needed

## Notes

- The website uses Dartmouth green (#004a37) as the primary color
- All pages share a common header and footer structure
- JavaScript functionality can be added to `scripts/main.js`
